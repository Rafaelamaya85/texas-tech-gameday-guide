# Project Scope Statement

## Project Name

Texas Tech Football 2026 Game Day Guide

---

## Project Purpose
**Why am I doing this project?**

The purpose of this project is to create a simple website that helps Texas Tech football fans who are traveling to Lubbock for a game. As someone who will be attending a game this fall, I wanted one place where I could quickly find everything I need to plan my trip. Instead of searching several different websites, this guide will combine the football schedule, stadium information, parking, restaurants, hotels, and game day tips into one resource. My goal is to make planning a Texas Tech football weekend easier for both myself and other fans visiting Lubbock.

## Description
**How would you describe this project?**

This project is a two-page website or guide built using HTML and CSS. The homepage will introduce the Texas Tech Football 2026 Game Day Guide and display the 2026 Texas Tech football schedule. The second page will provide helpful information for fans traveling to Lubbock, including Jones AT&T Stadium details, parking recommendations, nearby restaurants, hotels, and game day tips. The completed project will be published using GitHub Pages and stored in a professional GitHub repository along with all required project management documentation.
