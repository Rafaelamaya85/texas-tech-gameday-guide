# Texas Tech Football 2026 Game Day Guide

## Project Purpose

This project was created for my final project in CMPA 3301 at Texas Tech University. The goal is to provide Texas Tech football fans with helpful information for planning a trip to Lubbock for a home game.

The website includes the 2026 football schedule, stadium information, parking options, hotels, restaurants, and game day tips.
